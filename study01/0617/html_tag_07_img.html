html태그학습_07이미지태그

이미지태그는 웹문서에 이미지를 표시하기위해 사용하는 태그이며 source의 약자인 'src'속성과 이미지가 나오지않을때 대체할 문자'alt'속성을 사용한다.

웹에서 사용하는 이미지 종류에는 jpg(고화질사진), png(w3c에서만든 이미지포멧, 고화질, 투명배경지원,로고,아이콘), gif(단순한 색상 이미지만들때, 애니메이션, 아이콘)가 있으며, html5에서 추가된 svg이미지도 사용된다.

이미지태그는 다른태그와는 달리 닫는태그가 없다. img src="이미지경로나 주소"로 작성된다.

1. 이미지삽입하기

절대주소를 사용하여 이미지 삽입

상대주소를 사용하여 이미지 삽입

URL주소 - 웹서버에 위치한 파일이나 문서가 위치한 정확한 경로나 주소, 위치를 말한다.

주소는 절대주소와 상대주소가 있다.
절대주소는 http://~로 시작하는주소, 상대주소는 현재문서의 위치를 기준으로 주소가 결정된다.
./폴더명/파일명.확장자 =>문서를 기준으로 하위폴더일경우
../폴더명/파일명.확장자 =>문서를 기준으로 폴더가 다를때
파일명.확장자=>문서를 기준으로 같은 위치에 있을때

url 작성법 : http://도메인명/폴더명/파일명.확장자



2. 이미지주소(경로)가 잘못된 경우

alt 는 alternate text 의 약자로서 이미지가 나오지않을때 사용자가 이해할수있는 대체문소를 말한다. 시각장애인이 스크린리더를 통해서 음성으로 이해할 수 있는 문자임. 웹접근성을 위하여 반드시 삽입함. W3C HTML마크업 검사시 오류가나오기 때문에 반드시 사용해야한다.

3. 이미지크기지정속성-width(가로), height(세로)

->width,height 적어주지않으면 브라우저가 그 역할을 대신한다.(속도저하) 꼭 가로세로 써줘서 최적화시키기.

4. 반응형 이미지 만들기

반응형 이미지는 브라우저 가로폭을 줄이면 이미지도 같이 작아지며, 늘리면 반대로 이미지 크기도 커지는 기술. 가로세로가 비율유지되면서 자동으로 조정이된다.

이미지가없을때 이미지 대신하여 표시해주는 사이트 - placeholder.com

웹기획시 레이아웃 작성할때 이미지 제작하지않고 이미지처럼 가로세로크기로 표시해주는사이트이다.

html5에서추가도니 이미지 태그

이미지에 설명글을 넣기위한 새로운 태그가 추가되었다. figure, figcaption태그로 작성한다.
